WEB PUBLISHING SERVICES

프론트 엔드를 책임지는 웹 퍼블리셔 & UI 개발자

Responsive

다년간의 웹 퍼블리싱 경험을 바탕으로 N-Screen에 대응할 수 있는 기술적 노하우를 갖고 있으며, HTML5 문법에 맞춰 시맨틱한 구조와 논리적인 구조의 UI 화면을 설계합니다.

HTML5 APP

UI개발에 최적화된 리소스를 바탕으로 모듈화된 자바스크립트, 자바스크립트 프레임웍(백본JS, 앵귤러JS) 을 사용하여 프론트를 개발하고, NODE SERVER Module을 이용하여 백엔드까지 처리하고 있습니다.

Modulize

HTML, CSS, JAVASCRIPT를 모듈화 하여 집중 개발을 할 수 있도록 합니다. 또한 프론트/백엔드 개발 프로세스를 이해를 바탕으로 설계된 퍼블리싱 소스를 제공하여 백엔드 개발시에도 높은 능률로 개발이 가능합니다.

SKILLS

N스크린에 대응하는 웹 사이트 개발을 위한 HTML5 기반의 최적화된 코드를 제공합니다.

시맨틱한 HTML HTML5, 시맨틱, SEO

콘텐츠의 구조화, 검색 엔진 최적화에 초점을 두고, PC, MAC, 타블릿, 스마트폰의 웹 브라우저(익스플로러부터 표준브라우저, 하이브리드 앱)에서 완벽한 화면을 보여주기 위해 완벽한 크로스 브라우징을 합니다.

웹 접근성 동일한, 동등한 UX

웹 접근성 마크 획득 경험을 토대로 웹 접근성 가이드 지침을 따를 수 있도록 사용자 모두에게 동일한 내용을 이해할 수 있도록 사이트를 개발 합니다. 또한 각 파트별로 기술적, 업무적 분담을 정확히 판단하여 원할한 프로젝트를 진행하도록 합니다.

CSS 모듈화 Vanilla CSS, SCSS, LESS

사용되는 스타일은 모듈화하여 표준 가이드를 설계하고, 모듈끼리 조합하여 화면을 스타일링 하여 효율적으로 집중 개발을 할 수 있도록 합니다. 또한 개발정책에 따라 SASS, LESS 기반의 CSS 전처리기를 이용하여 더욱 진보된 스타일링 작업을 할 수 있습니다.

부트스트랩 Response Web

부트스트랩 템플릿 디자인을 이용하면 모바일부터 PC의 웹까지 대응할 수 있는 반응형 웹 사이트를 제작할 수 있습니다. 특히 자바스크립트 프레임웍과 어플리케이션 개발 솔루션을 조합하면 부트스트랩 기반의 템플릿만으로도 만족스러운 사이트를 구축할 수 있습니다.

JS 프레임웍 angularJS, BackboneJS

RequireJS를 이용한 의존성 소스를 구성하여 인터렉션과 퍼포먼스 향상을 이뤘습니다. REST API를 통한 비동기 페이지 개발도 가능합니다. 또한 Vanilla JS를 이용하여 매우 경량화된 프론트 UI기능 구현을 할 수 있습니다.

Web CMS CMS & Node

웹 표준 초기부터 여러건의 웹 CMS를 제이쿼리를 이용한 UI개발과 마크업을 전담하였습니다. 최근에는 자바스크립트 기반의 백엔드 Node모듈과 CSS프레임웍인 부트스트랩, 자바스크립트 프레임웍(angular)을 이용하여 사용성이 좋은 웹 CMS를 개발할 수 있습니다.

소스 형상관리와 빌드 SVN, GIT, Gulp, Node

소스 형상관리툴(SVN, GIT)를 통하여 효율적인 개발과 함께 프론트 소스를 Gulp 모듈을 이용하여 최적화된 라이브 소스로 배포할 수 있게 되었습니다. 또한 클라이언트와 웹 서버간 부하를 최소화하여 사이트 방문자로 하여금 UX를 향상시킬 수 있도록 하고 있습니다.

프론트엔드 가이드 제작 파워포인트와 HTML

매 프로젝트 종료시 제출되는 운영 가이드 파워포인트 문서나 HTML 형태로 제작, 배포합니다. 배포된 운영가이드 문서는 사이트 운영시 소스 파편화 최소화와 효율적인 운영 업무를 진행할 수 있도록 하는 좋은 지침이 될 수 있습니다
